Project Name: Recipe Page

Tech Used: HTML, CSS, Google Fonts

Description:
A beginner-friendly recipe webpage showcasing a simple omelette recipe. This project focuses on structured HTML layout, semantic elements, and styling a recipe page with images, lists, and nutrition information. It was my first attempt at organizing content in a visually clear and readable format.

Key Features & Learnings:
 - Semantic HTML: Used headings, paragraphs, lists, and divs to structure the content.
 - Responsive Layout Practice: Created sections for ingredients, instructions, prep time, and nutrition.
 - Image Integration: Displayed a recipe image to enhance visual appeal.
 - Content Organization: Learned how to logically separate recipe info for readability.

How to Use:
1. Open index.html in your browser.
2. Scroll through the recipe to see ingredients, preparation steps, and nutrition information.
3. Enjoy!

Project Name: Contact Form

Tech Used: HTML, CSS, JavaScript

Description:
A fully interactive contact form that validates user input before submission. The form includes fields for first name, last name, email, query type (radio buttons), message, and a consent checkbox. Upon successful submission, a confirmation message is displayed. Real-time error handling guides the user to complete all required fields correctly.

Key Features & Learnings:
 - Implemented form validation for text, email, radio buttons, checkboxes, and textareas.
 - Applied regular expressions to validate email formatting.
 - Dynamically displayed/hid error messages next to invalid fields.
 - Stored submission state using localStorage to maintain feedback across page reloads.
 - Enhanced user experience with immediate feedback and a success message.
 - Practiced event handling, conditional logic, and DOM manipulation in JavaScript.

How to Use:
1. Open index.html in your browser.
2. Fill out all required fields: first name, last name, email, query type, message, and consent checkbox.
3. Click the Submit button.
4. If any field is incomplete or invalid, error messages appear guiding you to fix issues.
5. Upon successful submission, a "Message Sent!" confirmation message will appear.

Difficulty: Beginner-Intermediate (Frontend Mentor / personal project)

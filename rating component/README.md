Project Name: Interactive Rating Component

Tech Used: HTML, CSS, JavaScript

Description:
A simple interactive rating widget built as a frontend practice project. Users can select a rating from 1–5 and submit their choice, which then triggers a thank-you state displaying the selected rating. This project emphasizes DOM manipulation, event handling, and dynamic content updates without page reloads.

Key Features & Learnings:
 - Interactive Rating Buttons: Click a button to select a rating; active selection is visually highlighted.
 - Dynamic Content Update: Selected rating is displayed in the thank-you message.
 - State Switching: Seamlessly toggles between rating view and thank-you view.
 - First Frontend Practice: Reinforced understanding of event listeners, class toggling, and DOM element selection.

How to Use:
1. Open index.html in a web browser.
2. Click a rating button (1–5) to select a score.
3. Click SUBMIT to view the thank-you message with your selected rating.

Difficulty:
Beginner — Focused on JavaScript event handling and DOM manipulation.

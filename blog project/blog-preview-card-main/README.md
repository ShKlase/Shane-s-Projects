Project Name: Blog Preview Card

Tech Used: HTML, CSS

Description:
A simple, responsive blog preview card built with HTML and CSS. The layout mimics a blog article preview with an image, title, description, and author section. Focused on practicing CSS layout, spacing, hover transitions, and mobile responsiveness.

Key Learnings:
 - Practiced semantic HTML structure
 - Learned responsive design with @media queries
 - Applied hover effects and clean layout spacing

Difficulty: Beginner project from Frontend Mentor

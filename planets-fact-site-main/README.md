Project Name: Planet Facts

Tech Used: HTML, CSS, JavaScript, Bootstrap

Description:
An interactive planetary facts web application that dynamically displays information, statistics, and visual representations of all eight planets in the solar system. Users can switch between planets and explore different data views, including overview, internal structure, and surface geology. The app fetches planet data from a local JSON file and updates the display in real time without reloading the page. This project focuses on DOM manipulation, dynamic content rendering, and clean UI layout using Bootstrap utilities.

Key Features & Learnings:
 - Dynamic Data Fetching: Loaded planet information from data.json and updated the DOM seamlessly.
 - Interactive Planet Buttons: Clicking a planet updates all visuals, descriptions, and facts dynamically.
 - Three Info Views: Each planet includes Overview, Internal Structure, and Surface Geology sections with individual button controls.
 - Responsive Layout: Designed with Bootstrap flex and grid utilities for responsive alignment and spacing.
 - Reusable Event Handling: Attached event listeners programmatically for efficient reactivity.
 - Custom Color Highlighting: Each planet’s theme color dynamically applied to selected info buttons.

Practice Areas:
JSON data handling
Dynamic UI rendering
Event delegation
Bootstrap styling and layout organization

How to Use:
1. Open index.html in your browser.
2. Use the navigation buttons to select a planet.
3. Explore details such as Overview, Internal Structure, or Surface Geology using the info buttons.
4. The facts section at the bottom updates automatically with each planet selection.
5. Hover or click around to explore responsive behaviors.

Difficulty:
Intermediate — Combines structured HTML/CSS layout with JavaScript DOM manipulation and dynamic event-driven rendering.

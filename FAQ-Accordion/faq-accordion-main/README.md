Project Name: FAQ Accordion

Tech Used: HTML, CSS, JavaScript

Description:
A dynamic FAQ Accordion that allows users to expand and collapse questions to reveal answers. Each question toggles individually, with only one answer visible at a time. The project features smooth interactions, a clean layout, and responsive design for mobile and desktop screens.

Key Learnings:
 - Learned how to manipulate the DOM to show/hide elements dynamically.
 - Practiced event delegation and click handling for multiple interactive elements.
 - Improved CSS styling for state changes, such as active buttons and hidden content.
 - Gained experience designing accessible and responsive layouts.

 Difficulty: Beginner

Project Name: E-Commerce Product Page

Tech Used: HTML, CSS, JavaScript

Description:
A responsive and interactive e-commerce product page built following Frontend Mentor guidelines. Users can browse product images, view a product lightbox modal, select quantities, add items to a cart, and manage cart items. The page includes live UI feedback for cart updates and dynamic image switching.

Key Features / Learnings:
 - Implemented dynamic product image gallery with clickable thumbnails and lightbox modal.
 - Built increment/decrement quantity controls with live updates to the cart.
 - Managed cart logic dynamically, including empty states and total price calculation.
 - Learned event delegation and propagation handling to manage clicks outside the cart.
 - Practiced responsive UI design and styling for desktop and mobile layouts.
 - Enhanced user experience with hover effects, selection borders, and visual feedback.

How to Use:
1. Open index.html
2. Browse the product images and click thumbnails to update the main product image.
3. Click the main image to open the lightbox modal, and navigate images using arrows.
4. Adjust the quantity using the increment/decrement buttons.
5. Click “Add to Cart” to add the product to the cart.
6. Access the cart using the cart icon to view or remove items.

Difficulty: Intermediate (Frontend Mentor project)

Project Name: Memory Game

Tech Used: HTML, CSS, JavaScript, Bootstrap, Font Awesome

Description:
This is a dynamic memory game built with HTML, CSS, and JavaScript. Players flip over cards (or circles) to match pairs. The game supports multiple players, timer tracking, move counting, and customizable themes and grid sizes. It is inspired by the Frontend Mentor Memory Game challenge and designed to enhance frontend development skills.

Key Features & Learnings:
 - Implemented dynamic theme and grid selection to customize gameplay.
 - Built a multi-player turn system with score tracking.
 - Created a timer and move counter to track game progress.
 - Designed a responsive gameboard for different screen sizes using CSS and Bootstrap.
 - Developed a win detection system that ranks players based on matched pairs.
 - Practiced advanced DOM manipulation, event handling, and conditional logic in JavaScript.

Project Name: FML Social Share Card

Tech Used: HTML, CSS, JavaScript

Description:
A responsive social share card component that displays an article with an author section and a share button. On desktop, clicking the share button toggles a share panel with icons for Facebook, Twitter, and Pinterest. The design is visually inspired by modern blog/article layouts, focusing on clarity, minimalism, and user interaction.

Key Features & Learnings:
- Implemented a clean HTML structure with semantic containers for content, author info, and share functionality.
- Used CSS to style the card, author section, and social share panel, including responsive layouts.
Applied JavaScript to toggle the social share icons on desktop click.
- Practiced DOM manipulation with querySelector and classList.toggle.
- Learned how to make a simple interactive element without relying on external libraries.

Difficulty: Beginner

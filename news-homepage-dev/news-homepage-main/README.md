Project Name: News Homepage

Tech Used: HTML, CSS, JavaScript

Description:
A modern news website homepage layout with a top navigation bar and interactive content sections. Users can click on category buttons (Home, New, Popular, Trending, Categories) to highlight the currently active selection. The main section features a large header article with a “Read More” button, accompanied by a secondary “New” articles sidebar. Below are additional content highlights displayed in a visually engaging grid layout.

Key Features & Learnings:
 - Implemented interactive navigation buttons that highlight the active category using JavaScript.
 - Structured the page using semantic HTML and CSS Flexbox/Grid for responsive layouts.
 - Created a visually appealing content hierarchy with header images, sidebars, and multi-item grids.
 - Practiced event handling and dynamic class toggling in JavaScript.
 - Learned to manage multiple elements with the same class and ensure only one is active at a time.

Difficulty: Beginner

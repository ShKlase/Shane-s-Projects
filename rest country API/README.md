Project Name: Rest Countries API App

Tech Used: HTML, CSS, JavaScript, Bootstrap, FontAwesome

Description:
A dynamic web application that allows users to explore information about countries worldwide. Built using a local data.json file (or REST API data), the app includes search, filter, dark mode, and detailed country views, creating an immersive, interactive experience for geography enthusiasts or educational use. The project emphasizes DOM manipulation, API/data handling, and responsive UI design, making it a perfect showcase for frontend development skills.

Key Features & Learnings
 - Dark Mode Toggle: Switch between light and dark themes dynamically, affecting all UI elements including search bars, cards, and borders.
 - Search Functionality: Live search filters countries by name as the user types.
 - Region Filter: Dropdown menu to filter countries by region (Africa, America, Asia, Europe, Oceania, Antarctic).
 - Detailed Country View: Clicking a country card opens a detailed page showing population, region, subregion, capital, native name, currencies, languages, top-level domains, and border countries.
 - Border Country Navigation: Border countries displayed as clickable buttons to navigate to their detailed pages.
 - Responsive Design: Fully functional on mobile, tablet, and desktop using Bootstrap.
 - Dynamic DOM Manipulation: Countries are appended programmatically; expanded view dynamically updates.
 - State Management: Tracks dark mode and active filters; toggles UI appropriately.
 - Learned: Advanced JavaScript techniques including fetch, event delegation, dynamic element creation, class toggling, template literals, and array manipulation (map, find, forEach).

How to Use
1. Open index.html in a modern browser.
2. Browse the homepage to see all countries displayed as cards.
3. Dark Mode: Click the moon icon in the header to toggle between light and dark themes.
4. Search: Type a country name in the search bar to filter countries in real time.
5. Filter by Region: Click the dropdown to filter countries by continent/region.
6. Detailed View: Click on any country card to open its detailed information page.
7. Border Navigation: On the detailed page, click any border country button to navigate directly to that country’s details.
8. Back Button: Return to the homepage using the "Back" button in the expanded view.

Difficulty
Intermediate – Excellent when looking to improve dynamic DOM manipulation, API/data integration, filtering logic, and UI responsiveness.

Project Name: Tip Calculator

Tech Used: HTML, CSS, JavaScript

Description:
An interactive Tip Calculator that dynamically calculates the tip and total per person based on the bill amount, selected tip percentage, and number of people. It features responsive design, custom tip input, and live error handling for invalid values.

Key Learnings:

 - Implemented real-time DOM updates using JavaScript event listeners.
 - Practiced modular JavaScript by separating calculation logic into reusable functions.
 - Improved UI/UX with hover states, responsive grid layout, and input validation.
 - Gained experience debugging floating-point precision and user input handling.

How to Use:
1. Open index.html in your browser.
2. Enter the bill amount.
3. Select a tip percentage or enter a custom value.Input the number of people splitting the bill.
4. The tip and total per person will update automatically.

Difficulty: Intermediate (Frontend Mentor project)
